# AstroWind — CloudCannon Editor Guide

## Pages

Pages use a **page builder** pattern. Each page has a `content_blocks` array where you can add, remove, and reorder sections (widgets). Available widget types include Hero, Features, Content, Steps, Pricing, FAQs, Testimonials, and more.

To create a new page, click **Add** in any page collection and select a template.

### Layouts

- **PageLayout** — Standard pages with full header and footer
- **LandingLayout** — Simplified header for landing/marketing pages
- **MarkdownLayout** — Simple text pages (privacy, terms)

## Blog Posts

Blog posts support the **Content Editor** for rich Markdown/MDX editing. Posts can use snippets for embedding YouTube videos, Tweets, and Vimeo videos.

### Available Snippets

- **YouTube** — Embed a YouTube video by ID
- **Tweet** — Embed a Tweet by ID
- **Vimeo** — Embed a Vimeo video by ID
- **Logo** — Insert the site logo

## Site Settings

Edit global site settings (name, SEO, blog config, theme) in **Data > config.yaml**.

## Icons

Icons use the format `tabler:icon-name` or `flat-color-icons:icon-name`. You can select from existing icons or type a new icon ID.
