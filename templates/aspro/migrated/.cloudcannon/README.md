# Astro Starter Pro

Welcome to the Astro Starter Pro site. This guide explains how to manage your content.

## Pages

Edit your site pages from [Pages](cloudcannon:collections/pages). Each page is built from reusable content blocks that you can add, remove, and reorder.

Available block types:
- **Hero** -- large banner section with a title, tagline, description, and action buttons
- **Features** -- grid of feature cards with icons
- **Content** -- centered section with a title, description, and optional image
- **Content 2** -- two-column section with text, item list, image, and action buttons
- **Values** -- grid of value cards
- **Service List** -- grid of service cards with icons
- **Contact Form** -- the contact form

## Blog

Write and manage blog posts from [Blog](cloudcannon:collections/blog). Posts support the visual editor for inline editing and the content editor for focused writing.

When creating a new post, you'll start in the content editor. Existing posts open in the visual editor by default.

## Data

Manage shared site data from the Data section:
- **Navigation** -- add, remove, and reorder navigation links
- **Social Links** -- update social media URLs (Twitter, GitHub, Discord)

## Creating new pages

Click the **+ Add** button on the Pages collection to create a new page. Choose **Page** for a simple text page or **Page Builder** to assemble a page from content blocks.
