# Welcome to Astro Starter Pro

This site is managed through CloudCannon. Here's how to work with it.

## Quick Links

- [Pages](cloudcannon:collections/pages) — Edit your site's pages
- [Blog Posts](cloudcannon:collections/blog) — Write and manage blog content
- [Site Settings](cloudcannon:collections/data) — Update site name, navigation, social links, and icons

## Pages

Open **Pages** to edit the homepage, about page, services page, and contact page. Each page is built from reusable content blocks that you can add, remove, and reorder in the visual editor.

Available block types:
- **Hero** — Large banner with title, tagline, description, and action buttons
- **Features** — Grid of feature cards with icons
- **Content** — Centered text section with optional image
- **Content with Items** — Side-by-side layout with text, list of items, image, and action buttons
- **Service List** — Grid of service cards with icons
- **Values** — Grid of value/principle cards
- **Rich Text** — Freeform rich text content

To add a new block, click the **+** button between blocks in the visual editor.

## Blog

Open **Blog** to create and edit posts. New posts open in the content editor where you can write using the rich text toolbar. Existing posts can also be edited visually.

## Site Settings

Open **Data** to edit:
- **site-settings.json** — Site name, description, author, navigation links, and social media links
- **icons.json** — The list of icons available in dropdown menus throughout the site

## Creating New Pages

Click **+ Add** in the Pages collection to create a new page. Choose **Page** for a simple text page or **Page Builder** for a page assembled from content blocks. New pages will automatically appear at the URL matching their filename.
